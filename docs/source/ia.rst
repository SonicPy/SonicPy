Travel-distance analysis
========================

SonicPy: Travel-distance program is used to determine the sample thickness which corresponds to the ulstrasound 
wave travel distance (\ :math:`d`).
This chapter describes the :ref:`GUI basics <gui_im>` of the Image analysis software and provides an :ref:`example Workflow <ia_workflow>` for determining the sample thickness.
